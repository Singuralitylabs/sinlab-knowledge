# Sinlab Knowledge について

**Sinlab Knowledge** は、シンギュラリティ・ラボが提供する Web スキル解説サイトです。

## このサイトの方針

本サイトは **解説サイト** として構成しています。手順を追って手を動かすハンズオン形式は採用せず、**読み物として完結する解説** と **個別記法・コマンド・概念のリファレンス** を主軸にしています。

各テーマは **完全に独立** しており、推奨閲覧順や対象者の案内も行いません。興味のあるテーマから自由に読み始めてください。

## テーマ

- **Web基礎** — Markdown と Git を体系的に解説します
- **開発実践** — React / Next.js / Supabase / フォーム / Git実践 / デプロイを扱います
- **AI駆動開発** — Claude Code を中心に、AIツール全体像を解説します

## ライセンス・出典

本サイトのコンテンツの一部は、シンギュラリティ・ラボが公開している既存教材を再構成したものです。
